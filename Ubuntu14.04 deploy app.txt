1. 安装SSH服务
  sudo apt-get install openssh-server

2. 安装Python3.5
  (1) sudo add-apt-repository ppa:fkrull/deadsnakes
  (2) sudo apt-get update
  (3) sudo apt-get install python3.5
  // (4) sudo rm /usr/bin/python
  // (5) sudo ln -s /usr/bin/python3.5 /usr/bin/python
  附: Python3.5不在Ubuntu软件仓库中所以需要增加仓库
      如果无法识别add-apt-repository命令, 则需要进行如下操作:
      a. sudo apt-get install python-software-properties 
      b. sudo apt-get install software-properties-common 

3. 取消原本的 Python 3.4 ，并将 Python3 链接到最新的 3.5 上：
  sudo mv /usr/bin/python3 /usr/bin/python3-old
  sudo ln -s /usr/bin/python3.5 /usr/bin/python3
4. 安装Python3包管理工具: python3-pip
  sudo apt-get install python3-pip

5. 安装需要的Python库
  sudo pip3 install jinja2 aiomysql aiohttp
  附: 如果没有切换Python3链接的动作, 则pip3会把Python库安装在Python3.4上

6. 切换回来链接文件：
  sudo rm /usr/bin/python3
  sudo mv /usr/bin/python3-old /usr/bin/python3

7. 安装MySQL数据库服务
  sudo apt-get install mysql-server
  
8. 安装 Nginx：高性能Web服务器+负责反向代理；
  sudo apt-get install nginx
  
9. 安装 Supervisor：监控服务进程的工具；
  sudo apt-get install supervisor
  启动 Supervisor
  sudo supervisord
